Tableau Dashboard Project
📌 Project Overview

This project analyzes the Boston Real Estate Housing Market using Tableau to uncover key factors influencing property prices, neighborhood demand, property taxes, transaction efficiency, and buyer preferences.

The objective was to help senior management improve the House Recommendation System by identifying pricing trends, premium neighborhoods, customer segments, and operational improvement opportunities.

🎯 Business Problem

The Boston real estate market is highly competitive and influenced by multiple factors such as:

Location
Property Size
Number of Bedrooms & Bathrooms
Property Taxes
Commercial Establishments Nearby
Transaction Closing Time

As a Lead Data Analyst, the goal was to analyze housing data and provide actionable recommendations that can improve:

Property recommendation accuracy
Pricing strategies
Customer targeting
Market expansion decisions

📊 Key EDA Questions Answered
1. What is the distribution of sale prices in Boston?
Most properties are concentrated between $100K–$200K.
Sale prices are positively skewed.
Luxury properties exist but form a small segment.
2. How do bedrooms and bathrooms vary across neighborhoods?
Area M and HS have the highest number of bedrooms and bathrooms.
Smaller areas contain fewer and smaller properties.
Housing density varies significantly across locations.
3. Is there a correlation between property size and sale price?
Strong positive correlation observed.
Larger houses consistently command higher prices.
Interior square footage is a major pricing driver.
4. What is the typical property tax across neighborhoods?
Cambridge Pkwy and Memorial Dr have significantly higher taxes.
Affordable neighborhoods show lower tax burdens.
Property tax reflects neighborhood desirability.
5. How do commercial establishments impact prices?
Properties near small commercial establishments (RC = 1) generally have lower sale prices.
Buyers appear willing to pay a premium for quieter residential areas.

🧮 Calculated Fields Created
Price Per Sq Ft
[Sale Price] / [Interior (Sq Ft)]

Used to compare property values across neighborhoods.

Days to Close
DATEDIFF('day',[First Date],[Close Date])

Used to measure transaction efficiency.

⚙ Parameters Created
Top N Streets Selector

Allows users to dynamically choose how many streets appear in visualizations.

Bedroom Selector

Allows users to filter properties by preferred bedroom count.
